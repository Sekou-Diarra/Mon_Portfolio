Utribu (Réseau Social)

Solution digitale pour Utrillo — Réseau social mobile natif avec architecture Cloud-Native.

Projet scolaire Méthode Scrum GitHub Flutter Spring Boot MongoDB Atlas
Contexte

Demande de Carole DELOUSTAL

Sous la direction de Carole DELOUSTAL, responsable de branche, ce projet vise à créer une solution digitale de mise en relation pour les étudiants d'Utrillo. Le projet utilise une méthodologie Scrum via GitHub pour assurer un développement agile et itératif.

Ma Mission Spécifique

Gestion des Publications

Je suis chargé du développement du module de publications (Feed), conçu sur le modèle d'Instagram. Une contrainte majeure de sécurité est appliquée : seuls les administrateurs ont les droits de création, de modification et de suppression des publications.

Format Insta

Affichage de contenus textuels, images et vidéos courtes avec une UX fluide.

Accès Restreint

Implémentation du RBAC (Role Based Access Control) pour la gestion admin.

Périmètre

Fonctionnalités Clés

DEV

Backend & API (Spring Boot)

Développement des endpoints REST JSON pour le CRUD des posts, intégration des WebSockets pour le temps réel et sécurisation via JWT OAuth2.

DATA

Stockage S3 & MongoDB

Gestion asynchrone des médias (images/vidéos) via Amazon S3 (Signed URLs) et stockage des métadonnées dans MongoDB Atlas.

Roadmap

Phases de Développement

Phase 1
Infrastructure & Auth
Sprint 0 : Setup CI/CD, Dockerize Spring Boot, Auth JWT.
Phase 2
MVP Publications
Développement du Feed type Instagram, upload S3 et droits Admin.
Phase 3
Interactions & Temps Réel
Likes, commentaires et notifications push via Firebase (FCM).
