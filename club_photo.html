Club Photo

Site web de gestion pour un groupe de photographes — inscription, connexion et galerie de photos.

Projet scolaire WampServer PHPMyAdmin VSCode PHP MySQL
Contexte

Club Photo de Limoges

Il s'agit de mettre en place une interface web permettant de gérer les articles et les différentes photos proposées par les membres du club. Le site doit permettre l'inscription et la connexion des membres, ainsi que l'affichage et la gestion des galeries photographiques.

Déroulement

Étapes du projet

Partie 1
Structure du site
Création de l'arborescence dans l'explorateur de fichiers et ouverture dans VS Code.
Partie 2
Page d'accueil
Création de la page d'index avec inclusion des différents blocs via une méthode PHP.
Partie 3.1
Gestion des photos
Affichage de toutes les photos selon le type d'article, consultation des détails d'une photo.
Partie 3.2
Gestion des articles
Affichage des articles triés par date de parution, photos associées à chaque article, ajout de photos.
Bilan

Synthèse des réalisations

Pages du site

Création de l'ensemble des pages avec une structure PHP modulaire.

Base de données

Implantation et configuration de la base de données MySQL via PHPMyAdmin.

Révision du code

Nettoyage et optimisation du code source.

Tests

Tests des différentes fonctionnalités : inscription, connexion, galerie.
